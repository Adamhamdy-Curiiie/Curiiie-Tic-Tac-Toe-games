\doxysubsection{Detailed Description}
Exception thrown when an unsupported player symbol is used. 

This exception is raised when attempting to use a symbol that is not allowed in the game (e.\+g., symbols other than \textquotesingle{}X\textquotesingle{} or \textquotesingle{}O\textquotesingle{}).
