\doxysubsection{Detailed Description}
Diamond Tic-\/\+Tac-\/\+Toe game with Minimax AI. 

This file contains the declarations for a Diamond-\/shaped Tic-\/\+Tac-\/\+Toe game where players must form both a 3-\/line and 4-\/line in different directions to win. The AI uses Minimax algorithm with Alpha-\/\+Beta pruning for optimal play.
