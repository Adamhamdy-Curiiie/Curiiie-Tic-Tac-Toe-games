\doxysubsection{Detailed Description}
Game board implementation for 5x5 Tic-\/\+Tac-\/\+Toe variant. 

This class represents a 5x5 game board where the game ends after 24 moves (board full except center cell). The winner is determined by counting who has more three-\/in-\/a-\/row sequences (horizontal, vertical, or diagonal).

Inherits from \doxylink{class_board}{Board$<$char$>$} template class.
