\doxysubsection{Detailed Description}
Header file for 5x5 Tic-\/\+Tac-\/\+Toe game implementation with AI support. 

This file contains the class definitions for a 5x5 variant of Tic-\/\+Tac-\/\+Toe where the game ends after 24 moves and the winner is determined by who has more three-\/in-\/a-\/row sequences. Now includes Smart AI player support.
