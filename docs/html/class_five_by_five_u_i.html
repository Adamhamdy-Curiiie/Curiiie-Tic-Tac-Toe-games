Detailed Description

User interface handler for the 5x5 Tic-Tac-Toe game.

User interface handler for the game.

This class manages user interaction, including getting moves from players and creating player instances. Now supports Human, Smart AI, and Random AI players. Inherits from UI<char> template class.
