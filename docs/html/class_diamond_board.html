Detailed Description

Board class for Diamond Tic-Tac-Toe game.

Manages a 7x7 grid with a diamond-shaped playing area (37 cells). Implements win detection for simultaneous 3-line and 4-line formations in different directions. Includes Minimax AI with Alpha-Beta pruning.
