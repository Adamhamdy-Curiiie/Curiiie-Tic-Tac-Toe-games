Detailed Description

template<typename T>
class Ultimate_TTT_Board< T >

Board class for Ultimate Tic Tac Toe game.

This game consists of a 3x3 main grid where each cell contains a smaller 3x3 sub-grid. Players must win individual sub-grids to claim cells on the main grid. First to get 3-in-a-row on main grid wins.
