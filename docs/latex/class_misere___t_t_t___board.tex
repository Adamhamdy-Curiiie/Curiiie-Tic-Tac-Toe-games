\doxysubsection{Detailed Description}
Represents a 3x3 Mis{\ucr}re Tic-\/\+Tac-\/\+Toe board. 

In Mis{\ucr}re Tic-\/\+Tac-\/\+Toe, making 3-\/in-\/a-\/row means the player loses.
