\doxysubsection{Detailed Description}
Represents the Pyramid-\/shaped Tic-\/\+Tac-\/\+Toe board. 

The board has 3 rows and 5 columns. Certain cells are marked as invalid, forming a pyramid shape. Winning lines are predefined triples of cells.
