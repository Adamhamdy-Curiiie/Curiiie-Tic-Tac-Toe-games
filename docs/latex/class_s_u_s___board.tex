\doxysubsection{Detailed Description}
Represents the SUS game board (3x3). 

Players place \textquotesingle{}S\textquotesingle{} or \textquotesingle{}U\textquotesingle{} letters on the board to form S-\/\+U-\/S sequences. Sequences can be formed horizontally, vertically, or diagonally. Each player earns points for completing S-\/\+U-\/S patterns.
