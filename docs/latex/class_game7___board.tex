\doxysubsection{Detailed Description}
\doxylink{class_board}{Board} class for 4x4 moving token Tic-\/\+Tac-\/\+Toe. 

Manages a 4x4 board where players move their tokens to adjacent cells. Initial setup\+: Row 0 = O X O X, Row 3 = X O X O, middle rows empty. Win by forming 3 in a row horizontally, vertically, or diagonally. Includes Minimax AI with Alpha-\/\+Beta pruning for computer players.
