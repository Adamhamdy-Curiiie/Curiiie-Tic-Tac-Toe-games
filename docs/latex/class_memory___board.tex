\doxysubsection{Detailed Description}
Represents the Memory Tic-\/\+Tac-\/\+Toe board (3x3) with hidden cells. 

During the game, all cells show as \textquotesingle{}\#\textquotesingle{}. At the end, reveals all moves that were made.
