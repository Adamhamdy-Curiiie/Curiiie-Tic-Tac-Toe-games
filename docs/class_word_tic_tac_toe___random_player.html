Scans board for empty cells, randomly selects one, and assigns random letter

Note
Uses rand() for randomization - ensure srand() is called during initialization
Returns
Pointer to Move object with random empty cell and random letter (A-Z)
Return values
nullptrIf no empty cells available or board pointer is null

Algorithm:

  1. Scans entire 3x3 board for empty cells (' ')
  2. Stores all empty cell coordinates in vector
  3. Randomly selects one empty cell
  4. Randomly selects a letter A-Z
  5. Creates and returns Move object
Exceptions
Printserror to cerr if board is full or pointer is null
Note
Uses rand() for randomization (seeded in constructor)
Prints move to console for transparency

Caller must delete returned Move pointer
